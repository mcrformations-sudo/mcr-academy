Le fonctionnement de votre plateforme

Tout ce qu'il faut savoir pour gérer MCR Academy : les rôles, la création d'une entreprise cliente, le suivi et les attestations.

Comment c'est organisé

La plateforme repose sur 3 niveaux d'accès. Chacun se connecte au même endroit (connexion.html) et est dirigé automatiquement vers son espace :

👑

Vous (admin)

Toi et Fred. Vous voyez tout : entreprises, stagiaires, attestations. Vous créez les entreprises clientes.

Code : MCR-PATRON
🏢

Le RH / patron

Le client. Voit uniquement sa société : ses salariés, leur avancement, leurs attestations.

Code RH propre à sa société
🎓

Le salarié

Suit ses formations et obtient son attestation. Voit seulement son parcours.

Code stagiaire de sa société

Vos étapes au quotidien

1

Connectez-vous en administrateur

La première fois : sur la page de connexion, cliquez sur « Créer un compte » et utilisez le code MCR-PATRON. Vous arrivez sur la console. Fred fait pareil de son côté : vous partagez les mêmes données.

2

Créez une entreprise quand vous vendez une formation

Onglet « Entreprises » → saisissez le nom de la société → « Créer ». La plateforme génère automatiquement deux codes : un code RH (pour le patron) et un code stagiaire (pour les salariés).

3

Distribuez les accès

Envoyez au patron / RH : le lien de connexion + son code RH + le guide entreprise. Le code stagiaire, lui, sera distribué par le RH à ses salariés (ou par vous si besoin).

4

Suivez l'activité

Onglet « Stagiaires » : toutes les personnes inscrites, toutes entreprises confondues. Onglet « Attestations » : toutes les attestations délivrées, avec export CSV pour vos archives.

5

Générez une attestation en PDF

Dans « Attestations », cliquez sur une ligne → l'attestation s'affiche → « Imprimer / PDF ». Pratique si un employeur la réclame et que le salarié ne l'a plus.

📋 Qualiopi : toutes les attestations sont conservées dans la base et exportables en CSV — pratique pour justifier le suivi sur les 5 ans demandés.

Bon à savoir

Confidentialité : un RH ne voit jamais les autres entreprises, un salarié ne voit que son parcours. Vous deux êtes les seuls à tout voir.
Mise à jour : quand on améliore la plateforme, c'est instantané pour tout le monde — personne n'a rien à réinstaller.
